#安装

1. Live2d插件 
    ```
    npm install --save hexo-helper-live2d
    ```
1. Gitalk评论插件 
    ```
    npm i --save gitalk
    ```
1. Sitemap站点地图 
	```
	npm install hexo-generator-sitemap --save
	npm install hexo-generator-baidu-sitemap --save
	```
1. 站内搜索
    ```
    npm install hexo-generator-searchdb --save
    ```
